# chen-yiwei · 陈奕蔚会计准则实务专家 Skill（合并版）

> 以**陈奕蔚**（会计视野论坛版主、中审众环会计师事务所合伙人）的思维框架为主框架，嵌入式整合**会计视野论坛历史答疑实时检索**能力。
> 工作流：**先检索陈版主历史问答 → 再查准则原文（本地缓存）→ 最后用陈版主心智模型综合回答**。

---

## 这是什么

本 Skill 把 AI 设定为「陈版主」，用于会计准则与审计实务的中文问答。它不是凭空生成答案，而是按三段式研究协议工作：

1. **检索历史答疑（BBS）** —— 先查会计视野论坛中陈版主是否已有相关答疑，命中则直接引述原话 + 原帖链接作为首要依据；
2. **查阅准则原文** —— 读本地缓存（CAS 1–42、准则解释、监管规则适用指引、案例解析等），未命中可在线抓取；
3. **综合判断** —— 用陈版主 5 个心智模型 + 12 条决策启发式研判，最后以「陈版主表达 DNA」输出。

集成了两个上游 Skill 的能力：
- `chen-yiwei-perspective`：陈版主思维框架（角色扮演 + 准则解读）
- `chenyiwei-bbs`：会计视野论坛历史答疑检索

## 安装方法

### 方式 A：WorkBuddy 导入
1. 下载仓库（或本仓库的 `chen-yiwei-skill.zip`），确保根目录有 `SKILL.md`；
2. WorkBuddy → 左侧「专家 / 技能」→「导入技能」→ 拖入 zip 或选择目录；
3. 建议勾选「非高风险自动安装」（本 Skill 已审计，纯角色/知识框架 + 只读检索）。

### 方式 B：手动放置（适用于 Claude Desktop / Cursor / 其他支持 Skills 的工具）
```bash
git clone https://github.com/Mrfeng2023/chen-yiwei-skill.git
# 把 chen-yiwei-skill/ 整个目录复制到你的 skills 目录
cp -r chen-yiwei-skill ~/.workbuddy/skills/chen-yiwei
```
纯聊天模型（网页版 ChatGPT / Claude.ai 等不认 zip）可把 `SKILL.md` 正文粘进「自定义指令 / 系统提示」中，角色设定与心智模型部分通用。

## 诚实边界

- 本 Skill 是基于陈版主 2003–2026 年约 3000 万字答疑提炼的**角色扮演 + 心智模型**，并非真人；
- 对陈版主从未讨论过的全新问题，仅能基于启发式推断，不保证与真人判断一致；
- 税务、法律、估值等非会计领域推断能力有限；
- **重要事项请执业人员自行把关**，本 Skill 不能替代专业判断。
